## 📋 5分钟快速部署步骤

### 第1步：准备账号（2分钟）
1. **GitHub账号**：https://github.com/ （免费）
2. **Vercel账号**：https://vercel.com/ （免费）
3. **Railway账号**：https://railway.app/ （免费）

### 第2步：上传代码（1分钟）
1. 在GitHub创建新仓库
2. 上传项目代码到仓库

### 第3步：部署后端（1分钟）
1. 访问 https://railway.app/
2. GitHub登录
3. New Project → Deploy from GitHub
4. 选择项目仓库
5. 复制返回的URL（API地址）

### 第4步：部署前端（1分钟）
1. 访问 https://vercel.com/
2. GitHub登录
3. New Project → 选择仓库
4. Root Directory: `admin`
5. 复制返回的URL（管理后台地址）

### 第5步：部署大屏幕（1分钟）
1. 再次 New Project
2. Root Directory: `screen`
3. 复制返回的URL（大屏幕地址）

## 📱 微信小程序配置

### 修改服务器地址
编辑 `miniprogram/app.js`：
```javascript
globalData: {
  serverUrl: 'https://your-project.up.railway.app'  // 替换为实际地址
}
```

### 配置域名白名单
登录微信公众平台 → 开发设置：
```
request合法域名: https://your-project.up.railway.app
Socket合法域名: https://your-project.up.railway.app
```

## 🆘 常见问题解决

### Q1: Railway部署失败？
**A:** 检查package.json中的start命令是否正确

### Q2: Vercel部署失败？
**A:** 确保Root Directory设置正确（admin或screen）

### Q3: 小程序无法连接？
**A:** 检查域名白名单和serverUrl配置

### Q4: 如何自定义域名？
**A:** 在Vercel和Railway控制台添加自定义域名
