## 📋 必须完成的任务（按顺序）

### 🚀 第1步：上传代码到 GitHub
**状态：** ⏳ 待完成

**操作步骤：**
1. 打开浏览器访问你的 GitHub 项目页面
2. 点击 "Add file" → "Upload files"
3. 将文件夹内的所有文件拖拽上传
4. 填写提交信息，点击 "Commit changes"

**预计时间：** 2分钟

---

### 🌐 第2步：部署后端到 Railway
**状态：** ⏳ 待完成

**操作步骤：**
1. 访问：https://railway.app/
2. 点击 "Login with GitHub"
3. 点击 "New Project" → "Deploy from GitHub repo"
4. 选择你的年会系统仓库
5. 等待自动部署完成
6. 复制生成的 URL（类似：`https://xxx.up.railway.app`）

**预计时间：** 1分钟

---

### 🔧 第3步：更新小程序配置
**状态：** ⏳ 待完成

**操作步骤：**
1. 打开文件：`miniprogram/app.js`
2. 找到第 4 行：`serverUrl: 'https://your-project.up.railway.app'`
3. 替换 `your-project.up.railway.app` 为你的实际 Railway 地址

**代码示例：**
```javascript
// 修改前
serverUrl: 'https://your-project.up.railway.app',

// 修改后（假设你的Railway地址是年会系统-2024.up.railway.app）
serverUrl: 'https://年会系统-2024.up.railway.app',
```

**预计时间：** 30秒

---

### 📱 第4步：配置微信小程序域名
**状态：** ⏳ 待完成

**操作步骤：**
1. 登录微信公众平台：https://mp.weixin.qq.com/
2. 进入"开发" → "开发设置"
3. 在"服务器域名"中添加：
   - request合法域名：你的 Railway 地址
   - Socket合法域名：你的 Railway 地址

**预计时间：** 1分钟

## 📞 获取帮助

如果遇到问题：

### 🔧 常见问题解决
1. **Railway 部署失败**
   - 检查 package.json 中的 start 命令
   - 确保所有依赖都正确配置

2. **Vercel 显示空白页**
   - 检查 admin/vercel.json 配置
   - 查看 Vercel 部署日志

3. **小程序连接失败**
   - 检查域名白名单配置
   - 确认使用 HTTPS 协议
   - 检查 serverUrl 配置是否正确
