## ⚡ 3步完成部署

### 第1步：上传代码到 GitHub（2分钟）

**方法A：直接上传（推荐）**
1. 打开你的 GitHub 项目页面
2. 点击 "Add file" → "Upload files"
3. 将这个文件夹内的所有文件拖拽上传
4. 点击 "Commit changes"

**方法B：ZIP压缩上传**
1. 将整个文件夹压缩为 .zip 文件
2. 在 GitHub 点击 "Add file" → "Upload files"
3. 上传 ZIP 文件
4. 在上传页面勾选 "Extract files before committing"

### 第2步：部署后端到 Railway（1分钟）

1. 访问：https://railway.app/
2. 点击 "Login with GitHub"
3. 点击 "New Project" → "Deploy from GitHub repo"
4. 选择你的年会系统仓库
5. 等待部署完成（自动检测 Node.js）
6. 复制生成的 URL（类似：https://xxx.up.railway.app）

### 第3步：配置小程序（1分钟）

1. 找到文件：`miniprogram/app.js`
2. 找到这一行：`serverUrl: 'http://localhost:3000'`
3. 替换为你的 Railway URL：`serverUrl: 'https://xxx.up.railway.app'`

## 🆘 常见问题

**Q: Railway 部署失败？**
A: 检查 package.json 是否正确，确保有 "start": "node app.js"

**Q: Vercel 显示空白？**
A: 检查 admin/vercel.json 路径配置是否正确

**Q: 小程序连接失败？**
A: 确认域名白名单已配置，URL必须是 https

**Q: WebSocket 连接失败？**
A: 在 Railway 环境变量中添加 NODE_ENV=production
